# 🔊 Text-to-Speech Reader (Hotkey Based)

A simple Python tool that reads aloud any selected text on your screen using a natural AI voice.

---

## 🚀 Features
- Read text from **any application** (browser, PDF, notes, etc.)
- Uses **Google Text-to-Speech (gTTS)** for natural voice
- Trigger using a **custom hotkey**
- Lightweight and easy to use

---

## ⌨️ HOW TO USE ---
STEP 1 --> **Ctrl + c**

STEP 2 -->**Ctrl + Alt + Shift**

Run the speak.py file > Select any text → Press Ctrl + c  → Press the hotkey → It will be spoken aloud
